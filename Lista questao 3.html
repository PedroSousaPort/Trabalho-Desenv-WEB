Diferença de linguagens server side e client side

Linguagem Server side

Linguagens server side em resumo, são aquelas que são processadas no servidor antes de chegar ao browser do usuário, exemplos:

  • PHP
  • Python
  • Java



    • Linguagem Client Side

    Ao contrário da anterior, as linguagens ''client side'' são aquelas que o código é carregado pelo próprio navegador do usuário, assim sem a necessidade de um servidor, exemplos:

    • JavaScript
    • HTML
    • CSS
